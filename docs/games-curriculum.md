# Games & Curriculum Documentation

## Game Specifications

### 1. Pixel Pet Coder (Ages 5-7) - FREE
**Slug:** `pixel-pet-coder`
**Tracks:** Spark Cubs

**Features Required:**
- A cute pixel pet (dog/cat/bunny) that moves through a maze
- Block coding interface where kids join puzzle pieces to command movement
- Maze with 3 levels (easy → medium)
- Goal: Collect all paw prints to unlock the door
- Visual feedback: pet wags tail when happy, bounces when collecting items
- Simple controls: Up, Down, Left, Right arrow blocks
- Award stars at the end (1-3 stars based on time)

**Technical Specs:**
- Language: HTML5 + JavaScript (or Scratch-compatible block code)
- Grid size: 8x8 for Level 1, 10x10 for Level 2
- Pet sprite: 32x32 pixel art
- Paw print collectibles: 3-5 per maze
- Maze walls: random but solvable

**Additional Requirements:**
- Bright, colorful design (pastel background, vibrant sprites)
- Large buttons for small hands
- No text instructions needed - use icons
- Sound effects: cheerful music, collect "ding", success "ta-da"
- Restart button always visible

### 2. Typing Ninjas (Ages 6-8) - FREE
**Slug:** `typing-ninjas`
**Tracks:** Spark Cubs, Code Rangers

**Concept:** Typing Speed & Accuracy

**Learning Goals:**
- Learn home row keys (A, S, D, F, J, K, L, Space)
- Build finger memory for keyboard layout
- Improve typing speed through fun exercises
- Understand accuracy vs. speed balance

**Activities:**
- Unplugged: Keyboard coloring sheet
- Practice: Typed challenges with falling letters
- Game play: Beat the timer, earn ninja stars
- Creative: Design your ninja character

**Time:** 90 minutes
**Materials:** Keyboard printouts, Computers with keyboards

**Concept:** Block Coding & Sequencing

**Learning Goals:**
- Understand step-by-step instructions (sequencing)

- Practice problem-solving through maze navigation

**Activities:**
- Unplugged: Paper maze with arrow stickers
- Block coding: Join puzzle pieces to move pet
- Game play: Collect paw prints in 3 levels

**Time:** 90 minutes
**Materials:** Printed mazes, Arrow stickers, Computers/tablets
